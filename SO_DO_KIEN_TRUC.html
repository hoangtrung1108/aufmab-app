💾 Luồng 1: Save & Sync — bấm Save → GAS → Google Sheets

Từ bấm nút Save trên S2 → gom data từ IndexedDB → đẩy lên GAS WebApp → ghi vào bsGew sheet

sequenceDiagram
    actor User as Kỹ thuật viên (iPad)
    participant S2 as Screen 2 (Phòng Detail)
    participant IDB as IndexedDB (Local)
    participant GAS as GAS WebApp
    participant Sheets as Google Sheets
    participant Drive as Google Drive

    User->>S2: Bấm nút 💾 Save
    S2->>IDB: dbGetAll('dem_le') — lấy tất cả records
    IDB-->>S2: [{local_id, values[], nhom, grosse, ...}]
    S2->>S2: Lọc: pending + values > 0
    S2->>GAS: gsRun('serverPush', JSON.stringify({dem_le, phong_new, vat_lieu_new, anh_urls}))
    GAS->>Sheets: writeDemLe() — ghi từng row vào bsGew sheet
    Sheets-->>GAS: {success, mapped_ids}
    GAS->>GAS: Gán sheet_id cho từng record
    GAS-->>S2: {success: true, mapped_ids}
    S2->>IDB: Cập nhật sync_status = 'synced' + sheet_id cho mỗi record
    S2-->>User: Toast "Sync OK!"

    Note over S2,IDB: Ảnh pending
    S2->>IDB: dbGetAll('anh') — lọc pending có data
    S2->>GAS: gsRun('serverUploadImage', base64)
    GAS->>Drive: getOrCreateSubfolder() → saveFile()
    Drive-->>GAS: drive_url
    GAS-->>S2: {success, drive_url, filename}
    S2->>IDB: Cập nhật sync_status = 'synced' + drive_url

📋 Luồng 2: Mở Card → Nhập liệu → Tap +1/+5

Click card S1 → renderS2() build giao diện → user tap số liệu → lưu IndexedDB ngay lập tức

sequenceDiagram
    actor User as Kỹ thuật viên
    participant S1 as Screen 1 (Card List)
    participant S2 as Screen 2 (Phòng Detail)
    participant IDB as IndexedDB
    participant MR as buildMR() — Material Row

    User->>S1: Click vào card (phòng + Gewerk)
    S1->>S1: openCard(card) — set curPhong
    S1->>S2: showScreen('s2')
    S2->>IDB: dbGetAll('dem_le') — lọc theo ma_phong + nhom
    IDB-->>S2: [records grouped by nhom → grosse]
    S2->>S2: renderS2() — build DOM theo Gewerk / DN / material rows
    S2->>MR: buildMR(rec) — từng cấu kiện
    MR-->>S2: DOM row với +5, +1, ô công thức, he_so

    User->>S2: Tap nút +1 hoặc +5
    S2->>IDB: dbGet('dem_le', local_id)
    IDB-->>S2: record hiện tại
    S2->>S2: pVal() — push giá trị vào values[]
    S2->>IDB: dbPut('dem_le', {...record, values:[...+val], sync_status:'pending'})
    IDB-->>S2: OK
    S2->>S2: Cập nhật DOM: hiện tổng mới trong ô mr-box
    Note over S2: Không re-render toàn bộ — chỉ update DOM element tại chỗ
    S2-->>User: Vibrate 30ms (haptic feedback trên mobile)

➕ Luồng 3: Thêm cấu kiện mới → Local + GAS + CONG_VIEC Sheet

Từ menu ··· → thêm cấu kiện mới → tạo IndexedDB records cho tất cả DNs + ghi vào CONG_VIEC

sequenceDiagram
    actor User as Kỹ thuật viên
    participant CTX as Context Menu (···)
    participant BSM as Bottom Sheet bsNewMat
    participant IDB as IndexedDB
    participant GAS as GAS WebApp
    participant CW as CONG_VIEC Sheet

    User->>CTX: Tap ··· trên Gewerk header
    CTX-->>User: Menu: Xóa / Thêm DN mới / Thêm cấu kiện mới / Nhân bản
    User->>CTX: Tap "Thêm cấu kiện mới"
    CTX->>BSM: openNewMatSheet(gew) — hiện form nhập tên + đơn vị
    User->>BSM: Nhập tên cấu kiện (VD: "Flansch DN200")
    User->>BSM: Chọn đơn vị: Stk / m / m²
    User->>BSM: Tap "Thêm"

    BSM->>IDB: dbGetAll('dem_le') — lọc các DN của Gewerk này
    IDB-->>BSM: [distinct grosse values]
    loop Mỗi DN đang có
        BSM->>IDB: dbPut('dem_le', {local_id: uuid(), nhom: gew, grosse: dn, ten_vl_german: name, values:[], don_vi, kieu_tinh, sync_status:'pending'})
    end

    BSM->>GAS: gsRun('serverAddMaterial', {nhom, ten_vl_german, don_vi, kieu_tinh})
    GAS->>CW: Tìm hàng cuối cùng của cùng Gewerk (nhom)
    GAS->>CW: insertRowAfter() — ghi row mới (ID, STT, nhom, ten_vl_german, don_vi, kieu_tinh)
    CW-->>GAS: OK
    GAS-->>BSM: {success: true}

    BSM->>BSM: closeSheet + renderS2()
    BSM-->>User: Cấu kiện mới xuất hiện trong tất cả DN
